# NoirNest

NoirNest is a dark luxury creative studio website built with Next.js, TypeScript, and Tailwind CSS.

## Structure

```text
src/
  app/
    layout.tsx
    page.tsx
    globals.css
    about/page.tsx
    services/page.tsx
    projects/page.tsx
    contact/page.tsx
  components/
    layout/
    sections/
    ui/
  data/
  lib/
```

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Local Preview

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Deployment

The project is connected to Vercel. Pushing to `main` deploys the site automatically.
