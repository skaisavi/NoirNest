# NoirNest

NoirNest is a multi-page dark luxury interior design studio website built with Next.js, TypeScript, and Tailwind CSS.

## Routes

- `/`
- `/about`
- `/services`
- `/projects`
- `/journal`
- `/contact`

## Structure

```text
src/
  app/
    layout.tsx
    page.tsx
    about/page.tsx
    services/page.tsx
    projects/page.tsx
    journal/page.tsx
    contact/page.tsx
    globals.css
  components/
    layout/
    sections/
    ui/
  data/
  lib/
```

## Scripts

```bash
npm run dev
npm run lint
npm run build
```
